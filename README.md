# 🍏 NutriVision – Smart Food Recognition & Nutrition Analyzer

A modern AI-powered food recognition and nutrition analysis web application built using **React**, **TypeScript**, and **Tailwind CSS**.

This repository demonstrates a complete **Manual Software Testing** lifecycle, including requirement analysis, test planning, test design, test execution, defect reporting, regression testing, traceability, and final test reporting.

# 📌 Project Overview

NutriVision enables users to:

- 📷 Upload food images for AI-based nutrition analysis
- 🥗 View detailed nutritional information
- 🔍 Search foods manually
- 📈 Track daily nutrition intake
- 💧 Monitor daily water consumption
- 👤 Manage personal profile and fitness goals

The application uses mock AI services to simulate food recognition while maintaining a scalable architecture for future backend integration.

# 🐞 Defect Summary

During testing, the following defects were identified:

- User session not maintained after page refresh
- Incorrect user name displayed after successful login
- Another user's profile and history displayed after login
- Selected application language is not applied
- Profile allows saving with empty required fields

All defects were logged in **Jira**, verified after implementation of fixes, and successfully closed following regression testing.
